### 방법
1. `issue`에 원하는 문제를 템플릿에 따라 작성합니다.
  * 해결했으면 해결 한 캡쳐본을 이용하여 `해결했습니다.`와 함께 올립니다.
  * 본인이 마지막으로 해결했다면, `issue`를 닫아줍니다.
2. 해당 문제를 업로드합니다.
  * 깃 커밋 메시지 본문에 어떻게 풀었는지 내용을 넣어주세요.
